AOJ Command OS has been installed successfully.

After clicking Finish, you can:

  - Use the Start Menu shortcut "AOJ Command OS" to launch the native desktop app window.
  - Open http://localhost:8000 in any browser if you prefer browser mode.
  - Share http://YOUR_IP:8000 with tablets and phones on the same Wi-Fi network.

Desktop mode and browser mode can run at the same time (one local app window + multiple Wi-Fi browser clients).

To stop AOJ, close the desktop window or end the python.exe process in Task Manager.
